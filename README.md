# Secret Revealing

A web app that displays random secrets fetched from an API.

## Live Demo
https://secret-revealing.onrender.com/

> **Note:** The app is hosted on Render's free plan. If it hasn't been visited recently, it may take up to 60 seconds to load on the first request. Please be patient!

## Built With
- Node.js
- Express
- EJS
- Axios
